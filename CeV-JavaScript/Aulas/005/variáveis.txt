#COMENTARIOS:

O comentario é utilizado para explicar pedaços do código enquanto o mesmo é desenvolvido, 
comentarios são ignorados pelo sistema ao rodar o arquivo de código.

No javascript pode ser utilizado dois tipos de metodos para comentar, sendo esses o:

//;
    Esse é utilizado para comentar uma unica linha
/* */
    Esse é utilizado para comentar tudo que estiver entre /* e */
    

#VARIAVEIS

Variaveis são como caixas que guardam dados, as variveis so podem guardar um dado por vez, sendo que quando é tentado inserir outro dado na variavel, o dado anterior é substituido.

    um exemplo usado no video dessa aula foi: 
    
        Imagine uma vaga de veiculo em um estacionamento,
        essa vaga cabe apenas um carro por vez, sendo 
        impossivel colocar dois carros na mesma vaga.
