#COMENTARIOS:

O comentario é utilizado para explicar pedaços do código enquanto o mesmo é desenvolvido, 
comentarios são ignorados pelo sistema ao rodar o arquivo de código.

No javascript pode ser utilizado dois tipos de metodos para comentar, sendo esses o:

//;
    Esse é utilizado para comentar uma unica linha
/* */
    Esse é utilizado para comentar tudo que estiver entre /* e */
    

#VARIAVEIS

Variaveis são como caixas que guardam valores, as variveis so 
podem guardar um valor por vez, sendo que quando é tentado 
inserir outro valor na variavel, o valor anterior é substituido.

um exemplo usado no video dessa aula foi: 
    
    Imagine uma vaga de veiculo em um estacionamento,
    essa vaga cabe apenas um carro por vez, sendo 
    impossivel colocar dois carros na mesma vaga.
        
        
    uma variavel ser definida de qual o tipo de valor,
    sendo por exemplo:
        
        STRING;
            valores de caracteres, sendo definido com aspas duplas ou simples, uma string pode conter todos
            ou maioria dos caracteres, numeros, simbolos, numeros e etc.
        
    
O nome que uma variavel recebe chama-se 'identificador',
para definir um identificador sem erros e de forma corre
ta é 
necessario se adequar a algumas regras, como:

    - NÃO começar com números;
    - NÃO conter espaços;
    - NÃO usar palavras reservadas.
        invés disso use o '_'.
    
O identificador pode;
    
    - começar com letras, $ ou _;f
    - mesclar letras com números;
    - usar acentos e símbolos.
    
OBS: Vale ressaltar que uma boa nomeclatura de identificadores 
é bastante importante para uma boa organização para leitura e manutenção do código.
