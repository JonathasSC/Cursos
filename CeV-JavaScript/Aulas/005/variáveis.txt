Nessa aula foi explicado o funcionamento de variáveis, que podem
ser entendidas como espaços a serem preenchidos com dados
facilitando o acesso e a manipulação dos mesmos durante todo o 
código.

outro exemplo para melhor compreensão:

imagine que variáveis sejam vagas para estacionamento de veículos,
essa vaga so pode ser preenchida com um veículo, para colocar outro veículo,
obrigatoriamente o veículo anterior deve sair da vaga.


Existe diversos tipos de dados, esses são:

números inteiros;
números flutuantes;
booleanos;
strings;

no JavaScript os números
