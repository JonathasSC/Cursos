Nessa aula foi explicado o funcionamento de variáveis, que podem
ser entendidas como espaços a serem preenchidos com dados
facilitando o acesso e a manipulação dos mesmos durante todo o 
código.

outro exemplo para melhor compreensão:

imagine que variáveis sejam vagas para estacionamento de veículos,
essa vaga so pode ser preenchida com um veículo, para colocar outro veículo,
obrigatoriamente o veículo anterior deve sair da vaga.

para definir uma variável no JS basta digitar:

var _NOME_DA_VARIAVEL_ = _DADO_RECEBIDO_

Obs: O nome da variável é chamado de identificador
o identificador pode:

começar com letra, $ ou _
conter letras e números
conter acentos

o identificar não pode:

começar com números
não pode ser palavras reservadas
conter espaço ( invés disso use o _ )


Existe diversos tipos de dados, esses são
os mais comumente utilizados:

números inteiros;
números flutuantes;
booleanos;
strings;

No JavaScript os números inteiros e números flutuantes são
ambos tratados como 'numbers' ou números comuns.
